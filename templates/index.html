OOTPBot

OOTPBot is a neural network that predicts statistics using player ratings. The network is trained using over 5000 player seasons in a modern simulation of OOTP 23.

To use OOTPBot, simply input your player ratings into the fields below. The neural network will output that player's projected wRC+.
